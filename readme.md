# Calculator

Uma calculadora simples desenvolvida em C# como parte dos estudos de lógica de programação e fundamentos da linguagem.

## 🚀 Tecnologias

- C#
- .NET
- Visual Studio Code

## 📚 Funcionalidades

- Soma
- Subtração
- Multiplicação
- Divisão

## ▶️ Como executar

1. Clone este repositório:

```bash
git clone https://github.com/marceeldiaas/calculator.git
```

2. Entre na pasta do projeto:

```bash
cd calculator
```

3. Execute a aplicação:

```bash
dotnet run
```

## 📖 Objetivo

Este projeto foi desenvolvido para praticar os fundamentos da linguagem C#, incluindo:

- Variáveis
- Métodos
- Estruturas de decisão
- Estruturas de repetição
- Organização do código
